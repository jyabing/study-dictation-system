基础信息
先确定这条素材属于哪个课节、哪种技能、哪种训练方式。
题干内容
题干是“问题 / 引导语 / 显示给用户的内容”。
普通题型需要填写文字题干；说 / 看字朗读可以只上传题干图片，但文字题干和题干图片至少要有一个。
克漏字提示： 手动挖空时，在题干里用 {词} 标记空位；自动挖空时，题干不要写 {}，只写提示 / 上文 / 引导语即可，不要和“回答”写成一模一样的完整句。
有上传文件时，训练页优先使用上传音频；没有文件时，才回退到上面的 URL / 路径。
用于说 / 看字朗读训练。可放朗读标记图、重音图、连读图、停顿图等辅助图片。
有上传图片时，训练页优先显示上传图片；没有上传图片时，才回退到上面的图片 URL / 路径。
回答内容
这里填写用户最终需要生成并记忆的目标回答完整句子。听 / 说默认也会用这里做识别比对。
克漏字提示： 手动挖空时，这里填写被挖掉的词，按顺序一行一个；自动挖空时，这里填写完整句子，由系统自动生成空格。
听 / 说训练答对后，优先播放这里上传的回答音频；没有文件时，才回退到上面的 URL / 路径或回答 TTS。
仅用于听 / 说题。这里填写可接受的读音型答案或别名。每行一个。 命中后会判为“读音正确，但标准书写需要加强”。
克漏字配置
读-克漏字用于随着熟练度自动增加空格。

填写方法

克漏字建议二选一使用，不要在同一题里同时混用“手动挖空”和“自动挖空”。

  • 手动挖空：在“题干”里用 {词} 标记空位；在“回答”里按顺序填写被挖掉的词。
  • 自动挖空：题干不写 {}；在“回答”里填写完整句子，由系统根据下方配置自动生成空格。
注意：同一题不要同时“题干里写了 {}”又勾选自动挖空，否则后面容易分不清到底按手动空位还是按系统自动空位来出题。
  • 单空：
    题干：{祇園精舎}の鐘の声、諸行無常の響きあり。
    回答：祇園精舎
  • 多空：
    题干：{祇園精舎}の鐘の声、{諸行無常}の響きあり。
    回答：祇園精舎
    諸行無常
  • 词组整体空：
    题干：{諸行無常}の響きあり。
    回答：諸行無常
  • 连续多个小空：
    题干：{諸行}{無常}の響きあり。
    回答：諸行
    無常
规则：题干里有几个 {}，回答里就写几个答案，并且按出现顺序对应。
如果你想把“諸行無常”作为一个固定词组练，请写成 {諸行無常}；如果你想拆开练两个小词，才写成 {諸行}{無常}。
已启用自动挖空：请在“回答”中填写完整句子，题干里不要再写 {}。
选择题配置
每个选项都可以是文字或音频。音频选项可配置“答错后显示文字”。
识别配置
听 / 说都走“识别成文字后比对答案”的路线。TTS 语言用于自动朗读，识别语言用于识别用户说出的答案。
